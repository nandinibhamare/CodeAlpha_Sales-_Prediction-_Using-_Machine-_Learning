# Sales Prediction Using Machine Learning

# Project Overview
This project predicts product sales based on advertising expenditure on TV, Radio, and Newspaper.

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

# Features
- Data Cleaning
- Exploratory Data Analysis
- Linear Regression Model
- Sales Prediction
- Future Sales Forecasting
- Model Saving using Joblib

# Dataset
Advertising Dataset

#ow to Run
1. Install dependencies
2. Run:
   python sales_prediction.py

## Output
- Model Performance Metrics
- Sales Forecast
- Marketing Channel Impact Analysis
